## Note
- Folder file harus :
```
C:\xampp\htdocs\PengaduanMasyarakat
```
- Akses di web dengan :
```
http://localhost/PengaduanMasyarakat/public
```
- Password admin = 123

- Untuk nama database, url, dan lain - lain dapat dilihat lebih lengkap di :
```
C:\xampp\htdocs\PengaduanMasyarakat\app\config\config.php
```
